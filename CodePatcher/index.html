CodePatcher Application

Welcome! You've reached the information page for CodePatcher.

CodePatcher is a desktop application built with PyQt5 that helps you apply patch files (.diff / .patch) to your code files.

How to Run CodePatcher:
This application is designed to be launched from the LoLLMs Web UI.
Please do not try to run this index.html file directly to start the application.

Navigate to your LoLLMs Web UI interface, find the CodePatcher service/binding, and use its associated "Start Server" or "Launch" button to run the application correctly.

This HTML page serves only as a placeholder and provides instructions on how to launch the actual PyQt application.
